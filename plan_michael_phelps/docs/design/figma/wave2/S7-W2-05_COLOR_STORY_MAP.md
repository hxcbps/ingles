## Story by Route

- `Hoy`: activation and momentum.
  - Dominant: primary blue.
  - Support: cyan guidance.
  - Accent: amber milestone badge.

- `Sesion`: concentration and execution.
  - Dominant: midnight/blue contrast.
  - Support: controlled cyan progress indicators.

- `Cierre`: consolidation and completion.
  - Dominant: neutral card tones.
  - Support: success semantic states.

- `Evaluacion`: confidence and evidence.
  - Dominant: neutral/ink clarity.
  - Support: status-based feedback colors.

- `Modulos`: orientation and roadmap.
  - Dominant: neutral structure.
  - Support: active module highlight using primary palette.

- `Progreso`: data visibility and trajectory.
  - Dominant: clean slate hierarchy.
  - Support: trend color coding with semantic consistency.

## Professional Combination Rules

1. One dominant hue family per screen.
2. Secondary hue only for directionality and transitions.
3. Accent hue only for achievement/reward moments.
4. Avoid adjacent high-saturation surfaces competing for attention.
