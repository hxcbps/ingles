# Figma Design Skeleton Agents - Sprint 7 Wave 2

Last updated: 2026-02-12

## Goal

Construir calidad visual profesional y consistente usando 5 agentes especialistas con entregables Figma claros por esqueleto.

## Specialist Agents

1. Palette Architect
- Define sistema cromático base (core, neutral, semantic, accent).
- Entrega: paleta primaria/secundaria, reglas de uso y anti-patrones.

2. Button System Designer
- Define jerarquía de CTA (Primary, Secondary, Tertiary, Destructive, Ghost).
- Entrega: matriz de estados (default, hover, focus, active, disabled, loading) y contexto de uso.

3. Typography Director
- Define escala tipográfica, roles semánticos y ritmo vertical.
- Entrega: type ramp completa con aplicación por tipo de pantalla.

4. Motion and Interaction Auditor
- Evalúa y redefine transiciones, tiempos, easing y uso narrativo.
- Entrega: motion scorecard y catálogo de patrones permitidos/prohibidos.

5. Color Story Reviewer
- Evalúa combinaciones y narrativa visual de marca (qué emoción comunica cada bloque).
- Entrega: mapa de storytelling cromático por vista (`Hoy`, `Sesion`, `Cierre`, `Evaluacion`, `Modulos`, `Progreso`).

## Review Gates

- Gate A: Consistencia cromática global.
- Gate B: CTA hierarchy clarity.
- Gate C: Legibilidad y ritmo tipográfico.
- Gate D: Motion intencional, no decorativo.
- Gate E: Storytelling visual coherente con objetivos de aprendizaje.

## Merge Rule

No iniciar implementación Wave 3 hasta completar `S7-W2-06 Unified Design Merge` y pasar los 5 gates.
