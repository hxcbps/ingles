# Daily Session V2 (Sesame-first)

Framework obligatorio:
`Input -> Noticing -> Output1 -> Feedback -> Output2 -> Transfer`

## Lunes a sabado (120 min)
1. `15'` Retrieval engine
- Revisa errores D+1, D+3 y D+7.
- Activa tarjetas SRS del foco adaptativo.

2. `20'` Book module (instruccion explicita)
- Mini leccion + 8 microdrills.
- Enlace directo a modulo asignado.

3. `20'` Guided listening
- 3 pasadas: gist, detail, inference.
- Extrae 8 chunks reutilizables.

4. `35'` Speaking task (Sesame)
- Fluency round.
- Accuracy round.
- Repair round.

5. `10'` Pronunciation lab
- Pares minimos + stress + shadowing.

6. `20'` Writing transfer
- Output diario + correccion + reescritura parcial.

## Domingo (300 min)
1. `25'` Weekly diagnostics.
2. `45'` Listening profundo.
3. `30'` Reading long-form.
4. `100'` Speaking mock long run.
5. `50'` Writing long-form (260+ palabras).
6. `20'` Pronunciation clinic.
7. `30'` Weekly review + gate + adaptive plan.

## Reglas de ejecucion
- Sesame es interlocutor principal.
- Sin traduccion durante speaking.
- Correccion: max 1 foco primario por dia.
- Si no hay evidencia + score + metricas, no cuenta.

## Evidencia minima diaria
- Audio speaking.
- Writing del dia.
- Log KPI y metricas leading.
- Rubrica (0-3): fluency, accuracy, interaction, pronunciation.
