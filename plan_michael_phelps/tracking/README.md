# Repositorio de evidencias V2

Si no hay evidencia + score + metricas, la sesion no cuenta.

## Estructura
- `tracking/daily/YYYY-MM-DD/audio/`
- `tracking/daily/YYYY-MM-DD/writing/`
- `tracking/daily/YYYY-MM-DD/kpi/`

## Minimo diario
- 1 audio speaking.
- 1 writing del dia.
- 1 log KPI con rubrica y metricas leading.

## Revision semanal
- Actualiza `tracking/registro_general.md` cada domingo.
- Usa `tracking/templates/weekly_review.md`.
